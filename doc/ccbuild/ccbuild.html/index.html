ccbuild - A strict developer's build utility

Abstract

This document is a general usage manual to ccbuild. It will introduce general ways of using ccbuild. It will also explain ccbuild's behaviour in more words then the manual does. The newest version of ccbuild can be found at the http://ccbuild.sourceforge.net
